## Description
Woody is a modernized take on an ever-popular WordPress layout â€” the horizontal masthead with an optional right sidebar and four column footer widgetized area that works perfectly for blogs and websites. It has a harmonious fluid grid using a mobile-first approach, and impeccable polish in every detail. Woody will make your WordPress look beautiful everywhere.
